# Optimization & SciML
This repository contains a sequence of **self-contained Jupyter notebooks**
accompanying the document entitled "Introduction to optimization methods 
for training SciML models" by Alena Kopaničákova (Toulouse-INP/IRIT/ANITI) 
and Elisa Riccietti (ENS Lyon).

## Setup
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
jupyter lab
```
